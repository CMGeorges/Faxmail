# Faxmail UML Documentation

This folder contains source-controlled UML for the full project.

- `system-context.puml`: actors and platform boundaries
- `component-diagram.puml`: solution layers and dependencies
- `domain-class-diagram.puml`: core business entities
- `smart-intake-sequence.puml`: enterprise routing flow
- `deployment-diagram.puml`: SaaS deployment view

The existing `Faxmail.vpp` file can still be used in Visual Paradigm, but these text diagrams are easier to review and update in Git.
